## Setup hardhat
```sh
npx hardhat
```

## Install necessry dependencies
```sh
npm i --save-dev dotenv hardhat-deploy @nomiclabs/hardhat-ethers@npm:hardhat-deploy-ethers ethers
```

## Import hardhat deploy in `hardhat.config.js`
```javascript
require('hardhat-deploy`)
```
